## Installation

```sh
npm i true-window-size
```

## Usage

1. Import:
   ```js
   import windowSize from 'true-window-size';
   ```
2. Use:

   ```js
   const winSize = windowSize();

   const winHeight = winSize.y;
   const winWidth = winSize.x;
   ```
